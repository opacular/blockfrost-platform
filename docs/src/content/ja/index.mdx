# Blockfrost プラットフォーム

Blockfrost プラットフォームは、お使いの Cardano ノードインフラストラクチャを高性能な JSON API エンドポイントへと変換します。

リレーインフラストラクチャの収益化を目指すステークプールオペレーター (SPO)、分散型 API ネットワークへの参加に関心のあるノードオペレーター、信頼性の高いローカル環境を必要とする開発者など、Blockfrost プラットフォームは 2 つのデプロイ方法でさまざまなニーズに対応します。

- **フリートに参加する**: お使いのノードを Blockfrost クラスタに接続し、分散型ネットワーク (Icebreakers プログラムとも呼ばれます) の一部として API リクエストを処理します。
- **独立して実行する**: 開発環境やプライベートインフラストラクチャの要件に最適なかたちで、独自のインスタンスを完全に制御してデプロイできます。

これは [オープンソースの Blockfrost バックエンド](https://github.com/blockfrost/blockfrost-backend-ryo/) の後継となる新バージョンであり、本番ツールに期待されるシンプルなデプロイと運用を維持しつつ、エンタープライズグレードの機能を既存の Cardano インフラストラクチャにもたらします。
