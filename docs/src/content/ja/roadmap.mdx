# ロードマップ

Cardano ではロードマップを明確な時代区分に分けて構成するのが好まれており、Blockfrost でも同じアプローチを採用しました。

![Blockfrost のロードマップ](/roadmap.png)

### Icebreaker 時代

Icebreakers 時代では、SPO や他のノードオペレーターが集中型 Blockfrost クラスタに参加できるようになります。インフラストラクチャを共有することで報酬を得つつ、Blockfrost の分散化を前進させます。

![Icebreaker era](/icebreakers_era.png)

### Hopper 時代

Hopper 時代の目標は、従量課金型 (pay-per-use) の分散型 API 課金システムを実装することです。Icebreakers はマイクロペイメントチャネル経由で支払いを受け取り、顧客側も取引を行うためのペイメントチャネルを開設します。

![Hopper era](/hopper_era.png)

ソリタリーモードでプラットフォームがクラスタ外で動作する場合、顧客は直接プラットフォームとペイメントチャネルを開設できる想定です。

### Cayley 時代と Liskov 時代

これらの時代はまだ研究中で、コミュニティでのさらなる議論が必要です。
